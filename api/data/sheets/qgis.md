## Map View Navigation
| Command | Description |
|---|---|
| `Space` / `PageUp` / `PageDown` / Arrow keys | Map pan — move the map |
| (menu only) | Pan map to selection — pans the map to the selected element |
| `Ctrl+Alt++` or mouse wheel | Zoom in — zoom into the map |
| `Ctrl+Alt+-` or mouse wheel | Zoom out — zoom out of the map |
| `Ctrl+Shift+F` | Zoom full — zoom out to show the full map |
| `Ctrl+J` | Zoom to selection — zoom to the selected element |
| (menu only) | Zoom to layer — zoom to the selected layer |
| (menu only) | Zoom to native resolution (100%) |
| (menu only) | Zoom last — zoom to the previous zoom level |
| (menu only) | Zoom next — zoom to the next zoom level |

## Project Management
| Command | Description |
|---|---|
| `Ctrl+N` | New Project — create a new project |
| `Ctrl+O` | Open Project — open an existing project |
| `Ctrl+S` | Save — save the project |
| `Ctrl+Shift+S` | Save As — save the project under a new name |
| `Ctrl+Shift+P` | Properties — open the project properties |
| `Ctrl+P` | New print layout — opens the dialog to create a new print layout |
| `Ctrl+K` | Search — opens the search bar |

## Layer Management
| Command | Description |
|---|---|
| `Ctrl+L` | Data Source Manager — add a new layer |
| `Ctrl+Shift+N` | New GeoPackage layer — add a new GeoPackage layer |
| `Ctrl+Shift+V` | Add vector layer |
| `Ctrl+Shift+R` | Add raster layer |
| `Ctrl+D` | Remove selected layer |
| `Ctrl+1` | Toggle Layers panel |
| `Ctrl+2` | Toggle Browser panel |

## Analysis Tools
| Command | Description |
|---|---|
| `Ctrl+Shift+I` | Identify Features — identify features on the map by clicking on them |
| (menu only) | Select feature — select features by area or single click |
| `F3` | Select feature by value |
| `F6` | Open Attribute Table |
| `Shift+F6` | Open Attribute Table — selected features only |
| `Ctrl+F6` | Open Attribute Table — visible features only |

## Advanced Tools
| Command | Description |
|---|---|
| `Ctrl+Alt+T` | Processing Toolbox |
| `Ctrl+Alt+P` | Python Console |
