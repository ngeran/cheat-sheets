## Navigation
| Command | Description |
|---|---|
| `Middle Mouse Drag` | Orbit viewport |
| `Shift+Middle Mouse` | Pan viewport |
| `Scroll Wheel` | Zoom in / out |
| `Numpad 0` | Toggle camera view |
| `Numpad 1 / 3 / 7` | Front / side / top view |
| `Ctrl+Numpad 1/3/7` | Opposite (back / other side / bottom) view |
| `Numpad 5` | Toggle orthographic / perspective |
| `Numpad .` | Zoom to selected |
| `~` (Pie menu) | View axis pie menu |

## Selection
| Command | Description |
|---|---|
| `Left Click` | Select object / element |
| `Shift+Click` | Add to selection |
| `A` | Select all |
| `Alt+A` | Deselect all |
| `B` | Box select |
| `C` | Circle select |
| `L` | Select linked mesh under cursor |
| `Ctrl+L` | Select all linked to current selection |
| `Ctrl+I` | Invert selection |

## Object Mode
| Command | Description |
|---|---|
| `Tab` | Toggle Object / Edit mode |
| `G` | Grab (move) |
| `R` | Rotate |
| `S` | Scale |
| `X / Y / Z` | Constrain transform to axis |
| `Shift+D` | Duplicate |
| `Alt+D` | Duplicate linked (shares mesh data) |
| `Ctrl+A` | Apply transform (location/rotation/scale) |
| `Delete` / `X` | Delete selected |
| `Ctrl+J` | Join selected objects |
| `M` | Move to collection |

## Edit Mode (Mesh)
| Command | Description |
|---|---|
| `1 / 2 / 3` | Vertex / edge / face select mode |
| `E` | Extrude |
| `I` | Inset faces |
| `Ctrl+B` | Bevel |
| `Ctrl+R` | Loop cut |
| `K` | Knife tool |
| `F` | Fill / make edge/face from selection |
| `M` | Merge vertices menu |
| `Ctrl+E` | Edge menu (bridge, mark seam, etc.) |
| `Alt+Click` | Select edge loop |

## Modifiers & Modeling
| Command | Description |
|---|---|
| `Ctrl+1` … `Ctrl+5` | Add Subdivision Surface modifier at level 1–5 |
| `Ctrl+2` | Common shortcut for Subdivision level 2 |
| `Shift+A` | Add menu (mesh, curve, light, etc.) |
| `Alt+M` | Merge menu (in edit mode with selection) |
| `Shift+R` | Repeat last action |
| `Shift+Ctrl+Alt+C` | Set origin menu |

## Rendering & Output
| Command | Description |
|---|---|
| `F12` | Render current frame |
| `Ctrl+F12` | Render animation |
| `F3` | Search operator by name |
| `Z` | Shading pie menu (wireframe/solid/material/rendered) |
| `Shift+Z` | Toggle wireframe overlay quickly |

## General / Interface
| Command | Description |
|---|---|
| `Ctrl+Z` | Undo |
| `Ctrl+Shift+Z` | Redo |
| `Ctrl+S` | Save file |
| `Shift+Ctrl+S` | Save As |
| `N` | Toggle sidebar (transform panel) |
| `T` | Toggle toolbar |
| `Ctrl+Space` | Toggle maximize area |
| `F2` | Rename selected object |
